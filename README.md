# Bank_system_project
# 🏦 Банковская система на Python (многоуровневый ООП-проект)

Это комплексный учебный проект, реализующий **модульную банковскую платформу** с нуля. Проект разделён на 7 этапов, каждый из которых добавляет новый функциональный слой. 

**Текущий этап (день 1):** Создано **ядро системы** — абстрактный класс счёта, его конкретная реализация, валидация данных, статусная модель и пользовательские исключения. Всё это составляет фундамент для последующих этапов.

## 🧱 Этап 1. Реализовано: базовый счёт и операции

На данном этапе создана основа, на которой будут строиться все остальные компоненты:

### Ключевые классы

- **`AbstractAccount`** (абстрактный класс)  
  Задаёт контракт для всех счетов: `deposit(amount)`, `withdraw(amount)`, `get_account_info()`. Содержит общие поля: `account_id`, `owner`, `_balance`, `status`.

- **`BankAccountRaw`** (промежуточный класс)  
  Реализует валидацию входящих данных (тип баланса, поддерживаемая валюта, допустимый статус) и автоматическую генерацию короткого UUID для номера счёта, если он не передан.

- **`BankAccount`** (конкретный класс)  
  Содержит бизнес-логику операций:
  - Проверка статуса счёта (`_check_active`).
  - Пополнение и снятие с проверками суммы и достаточности средств.
  - Возврат информации о счете.

### Модель статусов
- `активный` — все операции разрешены.
- `замороженный` — операции запрещены (выбрасывается `AccountFrozenError`).
- `закрытый` — операции запрещены (выбрасывается `AccountClosedError`).

### Поддерживаемые валюты
`RUB`, `USD`, `EUR`, `KZT`, `CNY`.

### Собственные исключения
- `AccountFrozenError`
- `AccountClosedError`
- `InvalidOperationError` (для некорректных сумм и типов)
- `InsufficientFundsError` (для недостатка средств)

## 🧪 Демонстрация работы

Пример использования (находится в блоке `if __name__ == "__main__"`):

```python
from bank_account import BankAccount, AccountFrozenError

# Создание активного и замороженного счетов
active = BankAccount("Иванов Иван", 1000, status="активный")
frozen = BankAccount("Петров Петр", 2000, status="замороженный")

print(active.get_account_info())  # Счёт: ACC-... | Баланс: 1000 RUB | Статус: активный

# Попытка снять с замороженного счёта (будет перехвачена ошибка)
try:
    frozen.withdraw(500)
except AccountFrozenError as e:
    print(f"Ошибка: {e}")  # Операция невозможна. Счёт заморожен.

# Валидные операции
active.deposit(500)    # Пополнение
active.withdraw(100)   # Снятие
print(active.get_account_info())  # Баланс: 1400 RUB
